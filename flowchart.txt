In this flowchart:

-"Recording Started…" is printed to the console.
-The script initiates the sound recording using the sounddevice library.
-It waits for the recording to complete.
-The recorded audio data is written to a WAV file using the scipy.io.wavfile module.
-"Recording Finished" is printed to the console.
